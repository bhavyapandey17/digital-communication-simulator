# Digital Communication Signal Simulator

This project demonstrates basic digital communication concepts using Python.

## Output Waveforms
![Waveforms](waveforms.png)

## What it does
- Generates a digital square-wave signal
- Adds random noise to simulate a noisy channel
- Visualizes:
  - Original digital signal
  - Noise signal
  - Noisy digital signal

## Tools & Technologies
- Python
- NumPy
- Matplotlib

## How to Run
1. Install Python
2. Install required libraries:
   pip install numpy matplotlib
3. Run the script:
   python signal_analysis.py

## Outcome
The project helps understand the effect of noise on digital signals, a fundamental concept in digital communication systems.
